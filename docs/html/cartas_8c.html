void parse ( char *  query)

Trata os argumentos da CGI.

Esta função recebe a query que é passada à cgi-bin e trata-a. Neste momento, a query contém o estado que é um inteiro que representa um conjunto de cartas. Cada carta corresponde a um bit que está a 1 se essa carta está no conjunto e a 0 caso contrário. Caso não seja passado nada à cgi-bin, ela assume que todas as cartas estão presentes.
